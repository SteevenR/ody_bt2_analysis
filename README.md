# Kingshot Bear Trap Stats

Outil perso pour analyser les performances des joueurs sur l'événement **Bear Trap** de Kingshot, à partir de screenshots, et publier les stats sur une page web statique (GitHub Pages).

## Workflow

1. Prendre des **screenshots** des résultats Bear Trap et les mettre dans `./screenshots`.
2. Lancer le script d'analyse :

   ```bash
   python analyse_beartrap.py
